💱 Conversor de Moedas

Um projeto de conversor de moedas desenvolvido com foco em prática de lógica, manipulação de DOM e experiência do usuário.

📌 Sobre o projeto

O Conversor de Moedas permite que o usuário selecione:

Moeda de origem
Moeda de destino
Valor a ser convertido

E, ao clicar no botão, o sistema exibe o valor convertido conforme a cotação.

Além disso, o projeto conta com uma interface moderna e intuitiva, com destaque visual para o resultado da conversão.

🚀 Funcionalidades

✔️ Seleção de moeda de origem
✔️ Seleção de moeda de destino
✔️ Entrada de valor personalizada
✔️ Conversão em tempo real
✔️ Exibição visual do resultado com bandeiras
✔️ Interface responsiva e estilizada

🛠️ Tecnologias utilizadas
HTML5
CSS3
JavaScript
🎨 Diferenciais do projeto

Este projeto vai além de um conversor simples:

✨ Design moderno com destaque visual
✨ Melhor experiência do usuário
✨ Organização de layout e responsividade
✨ Prática de lógica de programação aplicada
